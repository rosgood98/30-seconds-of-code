Explain briefly what the snippet does.

- Checks if a given char is a vowel


```js
function vowelCheck(c) { 
  return "AEIOUaeiou".indexOf(c) !== -1 
}
```

```js
vowelCheck('a'); // 'True'
vowelCheck('c'); // 'False'
```
